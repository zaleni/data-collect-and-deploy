## ROS (Robot Operating System)

ROS 是一个用于机器人软件开发的系统，它提供了构建机器人软件所需的各种工具和库。请注意，它并不是一个操作系统，而是一个构建于 Linux 之上的中间价，用于连接机器人和上层应用软件的桥梁，作用类似一个操作系统。

主要特性有：

- 分布式：节点之间通过网络连接，可以分布在不同的计算机上。
- 模块化：节点之间通过消息传递通信，可以方便地添加、删除和修改节点。
- 可扩展性：ROS 提供了丰富的工具和库，可以方便地扩展和修改。
