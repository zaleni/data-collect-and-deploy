

本仓库包含方舟无限ARX X5和松灵piper的数据采集和部署代码：



#### 环境安装：

Ubuntu 20.04

ROS2 foxy (ARX) / ROS1 noetic (piper)

[ARX X5](https://github.com/ARXroboticsX/ARX_X5) sdk, ros 安装

piper sdk, ros 安装

[orbbec](https://github.com/orbbec/OrbbecSDK_ROS2)摄像头ros 安装



本仓库包含：

[arx_x5_collect](./arx/collect/readme.md) 方舟无限数据采集

[arx_x5_deploy](/home/pc3/deploy/openpi/src/openpi/training/config.py) 方舟无限真机部署

松灵piper采集，部署
